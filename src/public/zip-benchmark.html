Browser ZIP Benchmark

Dieser Test vergleicht seriell fflate gegen zip.js fuer denselben ZIP-Input. Gemessen wird bewusst nur der Browser-Lesepfad: ZIP oeffnen, Entries auflisten und alle relevanten FIT-/WOA-Entries nacheinander in Memory materialisieren.

Kein Parallelismus, keine Worker-Tricks in zip.js. So bekommen wir einen fairen ersten Bibliotheksvergleich.
